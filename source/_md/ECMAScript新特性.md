## 函数

### 条件式调用

在 ES2020 中，可以使用`?.()`来条件式的调用一个函数。这和对象的条件式访问类似，在正常情况下如果直接访问一个不存在（nulll 或 undefined）的表达式，会抛出 TypeError。而使用条件式调用，在这种情况下，则整个表达式的值为 undefined，不会抛出异常。

```js
const test = (num, fn) => {
  fn?.(num);
};
```

当然他们也有一些些小小的区别：

```js
test.fn(); // 常规属性访问，常规调用
test?.fn(); // 条件式属性访问，常规调用
test.fn?.(); // 常规属性访问，条件式调用
```
